## 📋 Key Compliance Features

### Data Collection & Usage

| What We Collect | Why                   | Storage      | Deletion             |
| --------------- | --------------------- | ------------ | -------------------- |
| Emails          | Newsletters, support  | Encrypted DB | On request (30 days) |
| Feedback        | Service improvement   | Encrypted DB | On request (30 days) |
| Usage metadata  | Performance analytics | Anonymized   | 60-90 days           |
| IP/Browser info | Analytics only        | Anonymized   | 90 days              |
| File data       | Processing only       | In-browser   | Auto-deleted         |

### Security Measures

- ✅ 256-bit SSL/TLS encryption for all transmission
- ✅ Files processed in-browser (never stored on servers)
- ✅ Automated file deletion after 60 minutes
- ✅ Encrypted database storage for user data
- ✅ Access controls for authorized personnel only
- ✅ No third-party data sharing
